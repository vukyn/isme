- Go 1.24 or higher

1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   cd isme
   ```

2. Install dependencies:
   ```bash
   go mod tidy
   ```

### Using Make commands

  ```bash
  make run
  ```

  ```bash
  go run main.go
  ```
